# Portfólio Thayná Visual Marketing

Projeto de portfólio/site pessoal desenvolvido para apresentar os serviços, criativos, planos e canais de contato da Thayná Visual Marketing.

O site foi criado com foco em uma identidade visual elegante, suave e profissional, utilizando tons bege e marrom, animações leves e layout responsivo.

## ✨ Funcionalidades

- Header com navegação interna
- Seção hero com apresentação profissional
- Seção de serviços
- Carrossel de criativos
- Seção de planos com animação
- Footer com redes sociais e informações de contato
- Layout responsivo para desktop e mobile
- Animações em CSS
- Paleta visual personalizada

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- Google Fonts
- SVG Icons
- CSS Animations
- Media Queries# portifolio-thaynavisual
